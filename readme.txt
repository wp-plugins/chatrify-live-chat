== Installation ==

1. Upload `CHATRIFY_APP_NAME` directory to Wordpress plugins directory (`/wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the 'Chatrify' menu on the left.
4. Follow the instructions.
