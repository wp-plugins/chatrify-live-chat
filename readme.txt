== Frequently Asked Questions ==

= Do I have to install any software on my server to get this working? =

No, Chatrify is a hosted livechat service. Simply configure the plugin and you're done!

= Which web browsers work best with this plugin =

Chatrify works best in the following environment: IE 6+(PC), Firefox 2 or later (Mac, PC, or Linux), Safari 2 or later (Mac), Google Chrome (PC, Mac).

= I have installed the plugin & followed the instructions but still don't see the widget on my site? =
If you still don't see your widget on the site, you can contact us at support@chatrify.com with your registered email account.

Alternatively, you can also manually include the script in the footer file ("wp-content/themes/default/footer.php" in your wordpress installation).

= I need more help understanding Chatrify =

We are happy to help! Visit our site at http://www.chatrify.com or leave an email for us at support@chatrify.com and we will help you with installation.
